Задача 2a: интеграл не зависит от путей интегрирования, вариационная задача не
имеет смысла.

Задача 2b: решение уравнения Эйлера--Лагранжа имеет вид $y(x)\equiv C=const$, и
не может удовлетворить граничным условиям.

Задача 2c: решение существует при одном значении параметра $a$.

\pagebreak
\centerline{\Large Вариант 1.}
	Во всех заданиях ниже предполагается, что поставлена простейшая
	вариационная задача
	\[
		V[y] = \int\limits_a^b{F\left(x, y, y'\right)}dx.
	\]
	{\bf Определение.} Уравнение 
	\[
		\frac{\partial F}{\partial y} - \dfrac{d}{dx}\dfrac{\partial F}{\partial y'} = 0
		\qquad \left(\text{в краткой форме } F_y - \dfrac{d}{dx}F_{y'} = 0\right)
	\]
	называется уравнением Эйлера--Лагранжа.
	\begin{enumerate}
	\item 	(1б.) Для заданных функционалов записать уравнение Эйлера--Лагранжа:
		\begin{enumerate}
	%		\item $	\displaystyle F\left(x,y,y'\right)=\sin{y'}$ 
	%		\item $	\displaystyle F\left(x,y,y'\right)=y^3y'^3$ 
			\item $	\displaystyle F\left(x,y,y'\right) = y^2 - y'^2$ 
			\item $	\displaystyle F\left(x,y,y'\right) = 2xy' - y'^2 + 3y'x^2$ 
		\end{enumerate}

	\end{enumerate}
	{\bf Теорема.} Если непрерывно дифференцируемая на отрезке
	$[a, b]$ функция $y_0(x)$, проходящая через две заданные точки,
	доставляет экстремум функционалу $V\left[y(x)\right]$, то она
	удовлетворяет также уравнению Эйлера--Лагранжа. Интегральные кривые
	уравнения Эйлера--Лагранжа называются экстремалями функционала.
	\begin{enumerate}[resume]
	\item 	(2б.) Найти экстремали функционалов: 
		\begin{enumerate}
			\item $	\displaystyle V\left[y(x)\right]=
				\int\limits_0^1{y'(x)}dx, \quad y(0)=0,
				y(1)=1$; 
			\item $	\displaystyle V\left[y(x)\right]=
				\int\limits_0^1{y(x)y'(x)}dx, \quad y(0)=0,
				y(1)=1$; 
			\item $	\displaystyle V\left[y(x)\right]=
				\int\limits_0^1{\left[y(x) + xy'(x)\right]}dx, \quad y(0)=0,
				y(1)=a$ (исследовать множество решений в
				зависимости от параметра $a$);
			\item $	\displaystyle V\left[y(x)\right]=
				\int\limits_1^2{x^3y'^2(x)}dx, \quad y(1)=5,
				y(2)=2$; 
			\item $	\displaystyle V\left[y(x)\right]=
				\int\limits_1^2{\frac{y'^3(x)}{x^2}}dx, \quad y(1)=1,
				y(2)=7$.
		\end{enumerate}
	\item[\addtocounter{enumi}{1}\theenumi*] (2б.) Найти экстремали функционала:
		\[
			\displaystyle V\left[y\right] = \int\limits_a^b
			{x^ny'^2}dx \to \text{extr}, \quad
			y(a) = y_a, y(b) = y_b.
		\]
		Доказать, что при $n \geqslant 1$ никакие две точки,
		находящиеся по разные стороны оси $Oy$, не могут быть соединены
		экстремалью этого функционала.
	\end{enumerate}
